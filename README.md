# 📊 Sales Analysis Dashboard (Excel)

## 📌 Overview
This project showcases an interactive sales dashboard built in Microsoft Excel to analyze revenue, profit, customer performance, and sales trends.

## ⚙️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Slicers & Filters

## 📊 Features
- KPI tracking (Total Revenue, Profit, Profit Margin, Quantity Sold)
- Revenue analysis by age group, location, and product category
- Customer and salesperson performance analysis
- Payment method distribution (Cash, Credit Card, Debit Card, Mobile)
- Interactive filters (Location, Age Group, Payment Type)
- Time-based analysis (Month-over-Month, Week-over-Week trends)

## 📸 Dashboard Preview
![Dashboard](dashboard.png)

## 🔍 Key Insights
- Certain age groups (30–74) contribute the majority of revenue
- Mobile and card payments dominate transaction methods
- Specific cities like San Antonio and New York generate high revenue
- Seasonal variations observed in monthly performance trends

## 🎯 Business Impact
- Helps identify top-performing customers and locations
- Enables better decision-making through trend analysis
- Supports revenue optimization by analyzing payment behavior

## 🚀 How to Use
1. Download the `.xlsx` file  
2. Open in Microsoft Excel  
3. Use slicers and filters to explore insights
